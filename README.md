# Windows 7 PHP 7 Dev Vagrant

## Purpose

Quickly spin up a Windows 7 environment for building PHP 7.

## Set up

* Download the Windows7-php7dev.box vagrant box
* Run: ```vagrant box add path/to/Windows7-php7dev.box```
* Clone this repo
* cd to this folder
* ```vagrant up```
